# slippi-flatpak

An unofficial flatpak for Project Slippi's fork of Ishiiruka. To install, run `sudo flatpak-builder --install --force-clean --install-deps-from=flathub gg.slippi.Ishiiruka gg.slippi.Ishiiruka.json`. If you encounter any issues, please reproduce them on the upstream appimage before reporting them upstream.
