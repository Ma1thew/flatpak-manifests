# multimc-flatpak

A flatpak for MultiMC5. During setup, set the Java path to `/app/jre8/bin/java` for Java 8, or `/app/jre/bin/java` for Java 16. To install, run `sudo flatpak-builder --install --force-clean --install-deps-from=flathub org.multimc.MultiMC5 org.multimc.MultiMC5.json`. If you do use this, please don't report any issues upstream.
